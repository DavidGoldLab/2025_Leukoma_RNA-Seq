# 2024_Leukoma_RNA-Seq

The code used to perform RNA-Seq Analysis are provided in file "0_Code"
Before running code, download the transcriptome file "Leukoma.transcriptome-corset.fasta" (WEBSTIE) and place in folder "3_Corset"
Before running code, unzip all files (e.g. "find -type f -name "*.gz" -exec gunzip {} +")
